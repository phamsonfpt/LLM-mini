🔗 Sequence Diagram End-to-End

sequenceDiagram
    actor User as 👤 Người dùng
    participant UI as 🖥️ Streamlit
    participant API as ⚡ FastAPI
    participant Cache as 💾 Redis
    participant Qdrant as 📦 Qdrant
    participant BM25 as 📝 BM25
    participant Reranker as ⚖️ Reranker
    participant LLM as 🤖 LLM

    rect rgb(20, 60, 40)
    Note over User,LLM: 📥 NẠP DỮ LIỆU
    User->>UI: Upload tài liệu
    UI->>API: POST /upload
    API-->>UI: task_id
    par Dual Index
        API->>Qdrant: Embed + Upsert
    and
        API->>BM25: Tokenize + Add
    end
    end

    rect rgb(13, 59, 102)
    Note over User,LLM: 🧠 HỎI ĐÁP RAG
    User->>UI: Đặt câu hỏi
    UI->>API: POST /ask/stream
    API->>Cache: cache.get(hash)
    alt Cache HIT
        Cache-->>API: Cached answer
    else Cache MISS
        par Song song
            API->>Qdrant: Semantic
        and
            API->>BM25: Keyword
        end
        Note over API: RRF → 15 chunks
        API->>Reranker: Rerank
        Note over Reranker: 15 → 5
        API->>LLM: Prompt
        loop 50ms
            LLM-->>UI: SSE stream
        end
        API->>Cache: Lưu cache
    end
    end

    rect rgb(45, 27, 105)
    Note over User,LLM: 📚 HỌC TẬP
    User->>UI: Quiz request
    API->>Qdrant: Scroll all
    API->>LLM: quiz.jinja2
    LLM-->>UI: QuizSet JSON
    end
